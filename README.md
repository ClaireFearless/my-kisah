# My Kisah

My Kisah adalah sebuah aplikasi web sederhana yang dibangun menggunakan TypeScript. Proyek ini dikembangkan sebagai bagian dari proses pembelajaran dalam membangun aplikasi web yang terstruktur, ringan, dan mudah dikembangkan.

## Demo

Aplikasi dapat diakses melalui:
https://my-ye-shunguang.vercel.app

## Teknologi yang Digunakan

* TypeScript
* HTML & CSS
* (Tambahkan jika ada, seperti: Vite, React, Tailwind CSS)

## Fitur

* Menampilkan konten berupa kisah atau cerita
* Antarmuka yang sederhana dan mudah digunakan
* Performa ringan untuk penggunaan dasar

## Instalasi dan Penggunaan

Clone repository:

```bash
git clone https://github.com/ClaireFearless/my-kisah.git
```

Masuk ke direktori project:

```bash
cd my-kisah
```

Install dependencies:

```bash
npm install
```

Jalankan aplikasi:

```bash
npm run dev
```

## Deployment

Aplikasi ini telah di-deploy menggunakan Vercel sehingga dapat diakses secara publik.

## Tujuan Pengembangan

Proyek ini dibuat untuk:

* Mempelajari penggunaan TypeScript dalam pengembangan web
* Meningkatkan kemampuan dalam menulis kode yang terstruktur
* Melatih proses deployment aplikasi web

## Pengembangan Selanjutnya

Beberapa pengembangan yang dapat dilakukan:

* Penambahan fitur input dan manajemen kisah
* Integrasi dengan database
* Sistem autentikasi pengguna
